# Internationalization (i18n)

IVIDS features a highly efficient, key-based translation system that works without page reloads.

## Supported Languages
IVIDS supports 22 languages:
- `en`: English (Default)
- `es`: Español (Spanish)
- `fr`: Français (French)
- `de`: Deutsch (German)
- `nl`: Nederlands (Dutch)
- `it`: Italiano (Italian)
- `pt`: Português (Portuguese)
- `ro`: Română (Romanian)
- `pl`: Polski (Polish)
- `da`: Dansk (Danish)
- `sv`: Svenska (Swedish)
- `no`: Norsk (Norwegian)
- `cs`: Čeština (Czech)
- `ar`: العربية (Arabic)
- `hi`: हिन्दी (Hindi)
- `id`: Bahasa Indonesia (Indonesian)
- `ja`: 日本語 (Japanese)
- `ko`: 한국어 (Korean)
- `ru`: Русский (Russian)
- `tr`: Türkçe (Turkish)
- `vi`: Tiếng Việt (Vietnamese)
- `zh`: 中文 (Chinese)

## Implementation

### JSON Structure
Translations are stored in `lang/*.json` using nested objects.
```json
{
    "nav": { "home": "Home" },
    "search": { "placeholder": "Search..." }
}
```

### Usage
- **HTML**: Use the `data-i18n` attribute.
  ```html
  <h1 data-i18n="app.name">IVIDS</h1>
  ```
- **JavaScript**: Use the `window.i18n.t()` helper.
  ```javascript
  const label = window.i18n.t('nav.home');
  ```

### Dynamic Application
The `applyTranslations()` function scans the DOM for `[data-i18n]` and updates `textContent` or `placeholder` attributes.
